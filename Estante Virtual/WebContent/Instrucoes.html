Instruções de Uso

Este aplicativo tem como intuito, ser apenas uma estante virtual (por enquanto).

Seu uso é bem simples, se todos os passos forem seguidos corretamente.

Banco de Dados

Para começar o banco de dados que você deve utilizar é o Oracle 11g. No pacote Services existe uma classe com o nome ConfiguraDB, nesta classe existe variaveis nomeadas "Usuario" e "Senha", apenas troque o valor para o da sua maquina, e pronto a conexão funcionara corretamente. Alem disso se faz necessario a instalação do Tomcat 8.5 para a exeecução correta do software. Caso exista qualquer dificuldade favor enviar um email com a duvida para: saphienas@gmail.com

Uso do software

O software cadastra um novo livro na sua estante atraves da pagina Novo Exemplar, nesta pagina os campos estão devidamente validados "Anti-Trollagem". Após você cadastrar um novo Exemplar, você podera consulta-lo atraves de da pagina Consulta Exemplar, podendo tambem remove-lo quando bem intender. Este aplicativo é totalmente responsivo, então suporta qualquer tamanho de tela.

Este aplicativo é simples, e esta é sua primeira versão (Rascunho), outras funcionalidades serão adicionadas posteriormente.
